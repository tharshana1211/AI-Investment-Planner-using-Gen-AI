# 🚀 AI Investment Planner using Generative AI

An intelligent **AI-powered Investment Planning Web App** that generates personalized financial strategies based on user inputs like income, savings, and goals.

Built using **Streamlit + Generative AI**, this project helps users understand how to invest their money smartly with clear explanations and visual insights.

# 📌 Features

✅ Personalized Investment Plan Generation  
✅ AI-based Financial Recommendations  
✅ Interactive Investment Growth Visualization  
✅ Market Trend Analysis  
✅ Simple and Attractive User Interface  
✅ Ask Anything Financial Assistant  
✅ Real-time Graph Visualization  


# 🧠 How It Works

1. User enters financial details:
   - Age  
   - Monthly Income  
   - Monthly Savings  
   - Investment Duration  
   - Financial Goal  

2. The system processes the data.

3. A Generative AI model creates a personalized investment strategy.

4. The results are displayed along with graphs and insights.
